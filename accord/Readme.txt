# Accordion plugin made by Hassan Ejaz #

Usage:
1) Add "accord" attribute to any element, you want to change into accordion.

Example:
<div accrord>This is some text. Lorem ispum dolor sit amet!</div>


2) You can add title to accordion using "accord-title" attribute.

Example:
<div accrord accrord-title="Some title">This is some text. Lorem ispum dolor sit amet!</div>


3) You can give your custom "id" to "accord-wrapper" using "accord-id" attribute.

Example:
<div accrord accrord-title="Some title" accord-id="my-custom-id">This is some text. Lorem ispum dolor sit amet!</div>


4) You can also give your custom "class/classes" to "accord-wrapper" using "accord-class" attribute.

Example:
<div accrord accrord-title="Some title" accord-id="my-custom-id" accord-class="my-class some-other-class">This is some text. Lorem ispum dolor sit amet!</div>
